# Run
1. Clone this repository, i.e `Order_Managerment` <br>
2. Create virtual environment <br>
    ```python
    python -m venv venv
    ```
    Activate the virtual environment <br>
    - For Windows: `.\env\Scripts\activate` <br>
    - For Linux: `source env/bin/activate` <br>
3. Install dependencies, `pip install -r requirements.txt` <br>
4. In the repository folder, i.e `Order_Managerment`, run `uvicorn app.main:app` <br>
    - If the error occurs, please double check the installation of requirements <br>
